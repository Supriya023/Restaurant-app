🍽️ Restaurant Food Ordering App

A full-stack web application that allows users to browse restaurants, explore menus, place orders, and manage their food preferences seamlessly. This project is designed to provide a smooth and user-friendly online food ordering experience.

🚀 Features

* 🔐 User Authentication (Login / Signup)
* 🍕 Browse Restaurants and Menus
* 🛒 Add to Cart & Place Orders
* 📦 Order Management System
* ⚡ Fast and responsive UI
* 🔗 Backend API integration

  🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express.js
* Database: MongoDB (Atlas)
* Tools: VS Code, Git, GitHub
